Odoo Signature Module
📌 Overview
The Odoo Signature module enhances the Purchase Order (PO) workflow by automatically embedding digital signatures into approved documents. This feature ensures compliance, improves traceability, and streamlines the approval process within Odoo ERP.

🚀 Features
Automatic PDF generation after the final approval of a PO.

Includes Created By and Approved By details with corresponding digital signatures.

Only the last approval is recorded for clarity and compliance.

Seamless integration with Odoo’s existing approval workflow.

🛠️ Requirements
Odoo ERP (compatible with version X.X – specify your tested version).

Users creating POs must have a digital signature configured in their profile.

🔧 Digital Signature Configuration
To configure a digital signature in Odoo:

Click the Letter Icon in the top-right corner.

Select My Profile.

Click on Digital Signature.

Choose one of the following options:

Draw: Create a suggested signature using the drawing tool.

Upload: Import an existing signature file.

📂 Workflow
User creates a PO in Odoo.

After the final approval:

A PDF is automatically generated.

The document includes the PO creator and approver details with signatures.

The signed PDF is stored and available for download or sharing.
